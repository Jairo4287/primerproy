Te situas en el directorio y podes hacer un click con el boton derecho y click en "Git Bash Here"

- git clone "https://github.com/Jairo4287/primerproy" (direcci√≥n del repositorio remoto)

Si haces cambios en el repositorio remoto y queres sincronizar con el local:

- git status

- git add .
Adiciona todos los cambios al Stagin area

- git commit -m 'este es un primer comit'
